Task: Phase Summary at Distance Δ

A phase summary is a list of all phases that arrive at a given distance. Such a summary is very useful in seismogram interpretation. To generate one, do the following:

  1. read the model from an .nd file using model=mkreadnd(pfad);
  2. call syn=mksynopsis(delta,h,model); or
    syn=mksynopsis(delta,h,model,phaselist); or
    syn=mksynopsis(delta,h,model,phaselist,'silent');where
    delta
    the epicentral distance, in degrees
    h
    the focal depth in km
    model
    a model structure containing the velocity structure
    phaselist
    This optional parameter is a string matrix (generated by strvcat) that contains a list of phase names. Only these phases will be evaluated. If this parameter is not given, a default list will be used.
    'silent'
    If this parameter is given, no screen output will be produced.
  3. Now syn is a syn-structure containing the phase summary.

NOTE that multiple occurences of phases are considered.

When called without output parameter, the travel time synopsis (epicentral distance, travel time and ray parameter) is written to the screen: 

>>mksynopsis(70,0,prem,'silent');

If you call MKSYNOPSIS without output argument, the rays of all phases are also plotted.
