Release Notes for Freeze 26.02.2004

The first version and already has release notes... But don't worry, these are mainly for me to keep track, since I made some changes since the Backup-Freeze of 09.02.2004.

  • A new Routine MKTTBOXDATA has been introduced. This routine returns the path to the "data" subdirectory of the TTBOX directory. This allows easy access to this directory without hard-wiring the path.
  • A bug in MKTTCURVES was fixed: ray parameters were computed from take off angles using P velocities only. The routine now uses the appropriate velocity.
  • Some unneccessary old code was removed.
  • Test with onset and TauP Toolkit have been added in the section on validation.
  • MKTIM4P allowed SS phases inside the core and PKP, PKIKP outside the inner core. That's repaired.
  • MKTTCURVES now also plots τ(p) curves.
  • MKCLR2MODEL now accepts a predefined list of depths at which to sample the model
  • Several modifications of the documentation have been made.
  • MKRAYP2ANGLE transforms ray parameters into take off angles at source depth.
  • MKX4P and MKTIM4P now handle depth phases recursively. MKSMARTTAKEOFF has been modified accordingly.
  • A page on phase names has been added to the doc files.
  • MKRAYFAN now accepts a color argument and thus allows to plot rays in arbitrary colors.
